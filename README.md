## About The Project

Initially created for a meta-heuristics course at UQAC, this web application can resolve optimization problems using the GWO algorithm.
For problems in two dimensions, the app can also display in live the algorithm execution.

### Built With

* [Vite](https://vitejs.dev/)
* [TypeScript](https://www.typescriptlang.org/)
* [VueJS](https://vuejs.org/)


<!-- GETTING STARTED -->
### Installation

1. Clone the project
   ```sh
   git clone https://github.com/killian-mahe/gwo-visualizer.git
   ```
2. Install the required packages and dependencies
   ```sh
   cd gwo-visualizer && npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```
